# Squat Form Classification using Machine Learning

This project presents an interpretable machine learning approach for classifying squat form using biomechanical features derived from pose estimation.

## Features
- Knee angle and trunk angle based feature engineering
- Classification using Random Forest, SVM, Logistic Regression
- Cross-validation evaluation
- Statistical significance testing (paired t-test)
