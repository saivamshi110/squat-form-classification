# Summary

This software provides an interpretable machine learning approach for squat form classification using biomechanical features such as knee angle and trunk angle.

# Statement of need

Existing deep learning methods lack interpretability. This project provides a simple, transparent, and reproducible approach using classical machine learning.

# Method

The system uses:
- Random Forest
- Support Vector Machine
- Logistic Regression

Evaluation is done using 5-fold cross-validation and statistical testing.

# Results

Random Forest achieved the best performance (~97% accuracy), showing strong classification capability.

# Conclusion

This project demonstrates that interpretable biomechanical features can effectively classify squat form.
